Connection Guide

Gmail / Google Workspace

You cannot use your normal password. You must generate an App Password.

  1. Enable 2-Step Verification in Google Account.
  2. Go to the App Passwords page.
  3. Create a new app (name it "MailFlow").
  4. Copy the 16-character code and paste it here.

Outlook / Hotmail

  1. Go to Microsoft Account Security page.
  2. Enable Two-Step Verification.
  3. Under App passwords, click "Create a new app password".
  4. Use this password in MailFlow.

Rate Limits (Important)

Using free providers to send thousands of emails will result in a ban. We recommend a >1 sec delay. Gmail limits to ~500/day. For bulk sending, consider SendGrid, Mailgun, or Amazon SES.
