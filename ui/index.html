Connection & Security Guide

Gmail / Google Workspace

Use App Password only. Regular account password is blocked by Google SMTP policy.

  1. Enable 2-Step Verification.
  2. Open App Passwords.
  3. Create app password named "MailFlow".
  4. Put generated 16-char password into App Password field.
  5. Set From Email equal to authorized mailbox address.

Outlook / Microsoft 365

  1. Enable multi-factor authentication in account security settings.
  2. Create an app password for SMTP submission.
  3. Use SMTP host/port from provider docs and verify mailbox SMTP AUTH is enabled.
  4. Keep delay >= 2s for cold accounts and first campaigns.

Anti-Block Checklist

  • Warm mailbox gradually: start with small daily volume.
  • Keep delay + jitter + batch cooldown enabled.
  • Use domain distribution to avoid bursts to one provider.
  • Use verified sender domain (SPF, DKIM, DMARC) whenever possible.
  • Avoid suspicious content patterns and too many links.
